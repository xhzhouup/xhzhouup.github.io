HelloWorld!关于Java起步阶段01

二月 7, 2020 发布在 Java

认识Java

一.快捷键

1.windows快捷键 Ctrl+C 复制 Ctrl+V 粘贴 Ctrl+X 剪切 Ctrl+A 全选 Windows+r CMD运行

2.IDEA快捷键 Alt+ins 快速生成代码（GET+SET方法，重写重载等……） Ctrl+/ 注释单行 Shift+F10 快速运行 Ctrl+alt+L 格式化代码 Ctrl+alt+向上箭头 复制当前行到上一行 Alt+向上箭头 当前行和上一行交换 Ctrl+鼠标左键 去到定义处

3.命令行 （win10下）windows+r?输入cmd 命令： 切换盘符 盘符： 查看目录 dir+目录 去到具体位置 cd+目录 返回上一层 cd.. 返回根目录 cd
清屏 cls 查看之前的命令 上下箭头 以树行显示进程 tlist -t 退出 exit 创建目录 md 目录名

4.jdk jdk: 开发者环境: jdk+jre (开发环境和运行环境) (java的基础类库+开发环境)javac.exe 程序员 jre: 运行环境 jre (java的基础类库) java.exe 用户

5.配置环境变量 拷贝jdk安装路径的bin目录(C:\Program Files\Java\jdk1.7.0_17\bin) 计算机右键 -属性- 高级系统设置- 环境变量 -系统环境变量

    (1)path 编辑: 粘贴路径

    (2)	新建:	JAVA_HOME:	C:\Program Files\Java\jdk1.7.0_17
        新建:	CLASSPATH:	.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar
        编辑:	path:		%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

编写源程序

  1. 第一次编程 1)步骤 创建一个.txt文件，然后把文件后缀名改为.java 2)编辑源码：创建类（java承载体）：class 类名{ //所有代码 } 3)程序入口：mian函数/方法 public static void main(String [] args){ //方法内容 } 4)编译运行：找到java源码文件所在位置，在cmd下让目录切换到该目录 编译：javac 文件名.java 运行：java 类名

5)注意事项 A:一个java文件可以创建多个java类，但只能有一个公共类（public） B:注释：//单行注释 /多行注释/ /** 文本注释*/ C:括号都是成对出现 D:所有标点必须用英文 E:严格区分字母大小写 F:类名、文件名用英文,不要以数字开头 G:用public修饰的类的类名必须和文件名一致 转移字符（\） \n ：换行 \r ：回车 \f ：换页符 \b ：退格 \0 ：空格 \s ：字符串 \t ：制表符 \”： 双引号 \’：单引号 \ddd： 八进制字符串(ddd) \uxxxx：16进制unicode字符串(xxxx)

  1. 标识符命名规则

范围：java中所有可以起的名字；如文件名、类名、变量名…… 组成：字母+数字+下划线+$ 规则：不能以数字开头；不能是关键字；类名首字母大写；多个单词要使用驼峰命名法 关键字：在java中有特殊含义的单词，全小写,48个

  1. 数据类型

1)基本数据类型 (1)整型：所有没有小数点的数 名称 关键字 字节数 范围值 字节型 byte 1（8bit） -128~127 短整型 short 2 -2^15~2^15-1 整型 int 4 -2^31~2^31-1 长整型 long 8 -2^63~2^63-1 注意：当定义的数据超过了int的范围，数值后面必须加上l(L) (2)浮点型：所有有小数点的数 单精度 float 4 3.402823e+38 ~ 1.401298e-45 数后面加f 双精度 double 8 1.797693e+308~ 4.9000000e-324 (3)字符型：所有字符 char 2 注意：只能是一个字符，并且该字符用单引号引起来 (4)布尔型：所有结果只产生正负 boolean 1 true|false 2)引用数据类型 类（class）：Object、String、Date…… 数组（Array） 接口（interface）：List、Map……

  1. 定义变量

语法1：数据类型 变量名； 变量名=值； 语法2：数据类型 变量名=值； 语法3：数据类型 变量名1，变量名2=值，变量名3； 注意：局部变量必须赋初值，否则在使用时会报错；全局变量有默认值

  1. 数据类型转换 精度排序：byte short char int long float double 注意：小转大，自动转；大转小，要强转，即赋值时前面加上括号写上要强转的类型 byte+byte=int；short+short=int; 赋值时，short=short+1出错 ；short+=1正确，+=能强转
那么我们可以利用EditPlus、Eclipse、IDEA等工具编写第一个项目代码

如果使用EditPlus或者txt文档编写的代码，需要win键+r键输入cmd打开dos命名面板进行编译和运行。 特别注意：编译时要带上文件后缀名，运行时不需要带文件后缀名。

class First{
	public static void main(String [] args){
		System.out.println("HelloWorld!");
	}
}
