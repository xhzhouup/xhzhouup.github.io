封装

1.概念：指类将自身的属性私有化，对外部提供公共方法来进行访问。

好处：

  • 隐藏实现细节，提供公共的访问方式。
  • 提高代码复用性。
  • 提高安全性。
public class fengzhuang {
    //私有化变量，对外提供get和set方法进行访问
    private String name;
    private int age;
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }
}

继承

概念：当前类派生出新的类，当前的类被称为父类/基类/超类(Object)，派生出来的类被称为子类/派生类，子类可以继承父类所有的属性和方法，并且可以扩展自己的属性和方法。

好处：

  • 简化代码，提高代码的复用性。
  • 提高了可维护性。

缺点：

  • 造成代码的耦合性过高。

重载与重写

1.重写(override):发生在继承中，当父类的方法不满足子类的需求，子类对该方法进行重新定义。
2.规则：

  • 方法名，返回值，参数列表必须一致。
  • 子类的访问修饰符的作用域可以大于等于父类。

3.重载(overload)：发生在一个类中，针对不同的情况写不同的方法。
4.规则：

  • 方法名一致。
  • 参数列表不同(个数，顺序，类型)。
  • 与修饰符、返回值无关。

super关键字

1.super发生在继承中，作用是调用父类的属性或者方法。
2.创建子类对象时，默认会先调用父类无参的构造方法，可以使用super指定调用父类的构造方法。
3.super和this不能同时存在于同一个构造方法中，因为这两个关键字都要放在第一行。

子类的构建过程

先调用父类对应的构造器实例化一个父类对象,然后实例子类,最后把两个关联起来,表示他们是父子关系。

代码执行顺序

父类静态 子类静态 父类属性-父类构造器-子类属性-子类构造器

1、什么继承？继承有什么优点？

继承是一个类在原有的基础上派生出新的子类。
优点：简化代码，提高代码复用性和可维护性。

2、继承使用什么关键字？

extends

3、一个类可以同时继承多个类吗？所有类的父类是什么？

不能同时继承多个类，所有类的父类是Object。

4、什么是重写？用什么方法可以强制检测当前方法是否重写父类方法？

重写是指当父类的方法不满足子类的需求时，子类对该方法进行重新定义。
使用@override可以强制检测。

5、什么是多态？多态主要的应用是什么？

多态是指用一个行为具有多个不同表现形式或形态的能力。
主要应用：

  • 向上转型：父类引用指向子类对象。
  • 向下转型：把一个子类类型的对象从原来的父类类型转换为自己原来的子类类型。

6、什么是抽象方法？什么是抽象类？Java抽象类可以有构造函数吗？

抽象方法是指当不知道该方法具体的实现是，用abstract修饰，这个方法为抽象方法。
抽象类不一定有抽象方法，有抽象方法的类一定是抽象类。
抽象类有构造函数，但是不能实例化，只能由子类来继承。

7、Java抽象类可以有static方法吗？abstract方法可以用static修饰吗?

可以有static方法，但是abstract和static不能同时修饰一个方法。

8、这个静态方法可以被重写吗？Java抽象类中可以包含main方法吗？

不能被重写，可以被调用；可以包含main方法。

9、oop的三个特性?

封装、继承、多态。

10、super是什么?应用场景?

super发生在继承中，作用是调用父类的属性或者方法。

11、继承的特点?继承的限制?

java中只有单继承和多重继承，子类不能直接访问父类的私有成员，只有拥有。

final修饰符

1.final修饰方法：该方法不能被重写。  
2.final修饰变量：该变量初始化后不能修改数值；成员变量必须要有初始值，局部变量尽量在代码块结束前赋值。  
3.final修饰类：该类不能有子类，不能被继承。

abstract

1.抽象方法：当不知道该方法的具体实现，这个方法可以用abstract修饰，这个方法为抽象方法，并且该方法不能有方法体({ });
2.抽象类：如果一个类中有抽象方法，那么该类是一个抽象类。

  • 有抽象方法的类一定是抽象类，抽象类不一定有抽象方法。
  • 其子类必须实现抽象类中的抽象方法，如果其子类为抽象类则没必要。
  • 抽象方法的访问修饰符不能为private，也不能被final和static修饰。
  • 抽象类不能被实例化。

3.抽象类作用：将拥有共同属性特征的类继承此类，父类中添加新方法时子类将全部拥有而不用逐一去新增，提高了代码的复用性。

多态

1.分类：

  • 静态多态(方法重载)：发生在一个类中，参数名相同，参数列表不同(个数，顺序，类型)，与返回值和修饰符无关。
  • 动态多态(方法重写)：发生在继承中，子类重写父类方法，返回值、方法名、参数列表必须一致，子类方法访问修饰符必须大于等于父类方法。

2.向上转型：父类引用指向子类对象。(把一个子类类型的对象转换成父类类型)

SuperClass sup=new SubClass();

3.向下转型：把一个子类类型的对象从原来的父类类型转换成自己原来的子类类型(一定向上转型过)。

SubClass sub=(SubClass) sup;

String类

1.length():返回字符串长度。
2.equals(String str):比较两个字符串的内容是否相同。
3.indexOf(String str):返回字符串第一次出现的下标。
4.toCharArray():把字符串转化为字符数组。

方法的重载

1.方法名一致
2.参数列表不同(个数，类型，顺序)
3.与修饰符、返回值无关

值传递和引用传递

1.值传递(pass by value)：指在调用函数的时候把实际参数复制一份传递到函数中，这样在函数中对参数进行修改影响不到实际参数。
2.引用传递(pass by reference)：指在调用函数的时候吧实际参数的地址直接传递给函数，这样在函数中对参数进行修改会影响到实际参数。

static

1.static用于修饰变量，方法和代码块。
2.静态变量(类变量)：指被static修饰的变量，能够被所有对象共享，当其中对象改变该变量值,其他对象也被改变,可以对象调用,也可以直接用类调用(可以不产生对象)。
3.实例变量：普通变量，对象独有变量,当它自身改变与其他对象无关,只能被对象调用。
4.静态方法(类方法)：可以直接被类调用(可以不产生对象)，该方法不能调用非静态属性，非静态方法和this。
5.静态代码块

    static{  
        //代码块  
    }

静态最优先执行，并且只会执行一次。
